你这傻B，学个屁php呀，还跟学java吧！！登录的部分： 表单用post提交用户名和密码，先假设你在数据存的密码是通过用户名和密码MD5后得来的，把表单提交过来的用户名和密码MD5后和数据库对比，如果一样，就成功啦，然后创建一个session在所有的页面来维持登录状态，Session关掉页面就没有了，这时候就可以用cookie了，用法就不说了。 看错了，关于文件组织的，看你用的什么框架应该有文档，都是MVC,八九不离十就是控制器，模型，视图，这些英文单词也很清楚啦
